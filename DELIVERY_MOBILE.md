## 技术实现

### Viewport 配置
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### 媒体查询断点

| 断点 | 用途 | 覆盖设备 |
|------|------|---------|
| 768px | 平板和小屏幕 | iPad mini, 小平板 |
| 480px | 大屏手机 | iPhone 14 Pro Max landscape |
| 430px | 标准手机 | iPhone 14 Pro Max |
| 375px | 小屏手机 | iPhone SE, iPhone 12/13 |
| 横屏 | 横向模式 | 所有设备横屏 |

### CSS 样式特点

- **单列布局优先**: 所有内容垂直排列
- **触控优化**: 最小 44px 触控目标
- **最大宽度限制**: 100% 宽度，无溢出
- **字体缩放**: 移动端稍小字体
- **颜色保持**: 紫色主题 (#4f46e5)

## 已知限制

### 设计限制
- 最小支持宽度: 360px
- 低于 360px 的设备可能出现布局问题
- 极小字体可能影响可读性

### 浏览器兼容性
- 需要支持媒体查询的现代浏览器
- IE 11 及以下不支持（已不符合实际需求）
